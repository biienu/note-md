# 主语

名词、代词、非谓语动词、从句可以作为**主语**



## 主语不能少

缺少主语怎么办？

1. 添加 `it` 作为主语，必须和天气有关
2. `there be` 作为主语
3. 被动
4. 人称代词 `i you we`



# 宾语
